# Hostel Dorm Allocation

1. Java console application for hostel room management.
2. Uses JDBC to connect with Oracle database.
3. Supports student allocation and vacating rooms.
4. Includes validation and vacancy handling.
5. Run `Main.java` after setting DB details in `DBUtil.java`.
